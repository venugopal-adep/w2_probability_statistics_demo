What is a Probability Distribution?

The probability distribution of a random variable describes the values that the random variable can take along with the probabilities of those values.

Discrete Probability Distribution

Uses a Probability Mass Function that provides the probability for each value of the random variable.

Continuous Probability Distribution

Uses a Probability Density Function that determines the probability within a given interval.
